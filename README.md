## 🔴 The Problem

Mahakumbh is the **world's largest human gathering** — attracting over **400 million pilgrims** across 45 days. This creates one of the most complex crowd management challenges on the planet.

| Challenge | Impact |
|-----------|--------|
| Uncontrolled crowd surges at ghats | Stampede risk, injuries, deaths |
| No real-time visibility for authorities | Delayed response to emergencies |
| Pilgrims have no route guidance | Confusion, congestion, long waits |
| Parking zones overflow without warning | Traffic gridlock across the city |
| Special events (Snan dates) multiply density 3–5× | Entire city becomes unmanageable |

> **Real consequence:** The 2013 Mahakumbh stampede at Allahabad Railway Station killed 36 people and injured hundreds — all due to unmanaged crowd flow.

---

## ✅ Our Solution

**KumbhFlow** is a real-time crowd intelligence platform that gives authorities and pilgrims a complete, live picture of crowd conditions across all major Mahakumbh zones — with AI-powered predictions, route optimization, and decision-support tools.

```
Pilgrim opens KumbhFlow
        ↓
Sees live heatmap of all ghats
        ↓
Gets AI route recommendation (safest, fastest path)
        ↓
Receives real-time alerts if their zone becomes critical
        ↓
Reaches destination safely ✅

Authority opens KumbhFlow
        ↓
Monitors all zones simultaneously on one dashboard
        ↓
Gets ML-based crowd forecast for next 8 hours
        ↓
Simulates emergency scenarios before they happen
        ↓
Deploys police/medical units proactively ✅
```

## ✨ Features

### 1. 🗺️ Live Crowd Heatmap
- Interactive SVG map of Prayagraj showing Ganga + Yamuna rivers
- 8 major zones rendered as color-coded circles
- 🔴 Red = Critical (70%+) · 🟡 Yellow = Moderate (40–69%) · 🟢 Green = Safe (<40%)
- Hover tooltips showing live pilgrim count, density %, and status
- Auto-refreshes every 30 seconds with simulated live data

### 2. 📊 Live Crowd Density Dashboard
- Real-time percentage bars for all 8 zones
- Sorted by density — highest risk shown first
- Animated fills with color transitions
- Instant visual scan for authorities

### 3. 📈 ML-Based Hourly Crowd Prediction
- Bar chart forecast from 6AM to 11PM
- Color coded by risk level
- Current hour highlighted with glowing border
- Based on historical Mahakumbh crowd patterns

### 4. 🧭 AI Route Optimization
- 3 optimized safe routes with full path details
- Shows estimated travel time and crowd level per route
- Best route auto-recommended with ★ marker
- Routes update based on live crowd conditions

### 5. ⚠️ Real-Time Alert System
- Scrolling live alert ticker at top of screen
- 3-tier severity: 🔴 Critical · 🟡 Warning · 🔵 Advisory
- Specific zone-based alerts with timestamps
- Broadcast button for authorities to push alerts

### 6. 🎛️ Crowd Simulation Engine
- 4 scenarios: Normal Day / Major Snan / Festival Peak / Emergency
- Time of day slider (6AM–11PM)
- Crowd multiplier control (50%–200%)
- Hourly timeline chart updates in real-time
- Zone-by-zone simulation cards with capacity bars

### 7. 📉 Analytics & Forecasting
- 7-day crowd trend line chart
- Peak hours forecast for next 7 days including special Snan dates
- Key metrics: avg pilgrims, prediction accuracy, response time
- AI-generated recommendations for authorities

### 8. 👮 Authority Command View
- Police deployment status per zone
- Medical units and field hospitals tracker
- Ambulance dispatch status
- Full incident log with severity, status, and responders
- One-click broadcast messaging to all pilgrims

### 9. 🤖 AI Chatbot Assistant
- Answers pilgrim questions in natural language
- Knows live crowd levels, safe routes, parking, timing
- Quick-prompt buttons for common questions
- Responds in context of current crowd data

## ⚙️ How It Works

```
┌─────────────────────────────────────────────────────────┐
│                    KumbhFlow Platform                    │
├─────────────┬──────────────────┬────────────────────────┤
│  Data Layer │  Intelligence    │  User Interface        │
│             │  Layer           │                        │
│  Simulated  │  ML Prediction   │  Dashboard (Pilgrims)  │
│  live crowd │  Engine          │  Authority View        │
│  data for   │                  │  Simulation Engine     │
│  8 zones    │  Route           │  Analytics             │
│             │  Optimization    │  AI Chatbot            │
│  Auto-      │  Algorithm       │                        │
│  refresh    │                  │  Mobile Responsive     │
│  every 30s  │  Alert           │  No install needed     │
│             │  Generation      │  Works offline         │
└─────────────┴──────────────────┴────────────────────────┘
```

### Key Technical Decisions
- **Single HTML file** — Zero dependency, works on any device, instant load
- **SVG Map** — Lightweight, scalable, interactive without any map API costs
- **Simulated live data** — Realistic crowd simulation with random walk algorithm
- **No backend needed** — All logic runs client-side for maximum reliability

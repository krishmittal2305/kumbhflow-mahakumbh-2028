# 🕉️ KumbhFlow — Intelligent Crowd Intelligence Platform
### Mahakumbh 2028 | Expert Hire × VIT Bhopal Hackathon

## 🚀 Live Demo
👉 [Click here to open live app](https://kumbhflow-2028.netlify.app)

## 📌 Problem Statement
Mahakumbh attracts 400M+ pilgrims. Unmanaged crowd movement causes
delays, safety risks, and operational failures. KumbhFlow solves this
with real-time crowd intelligence for authorities and pilgrims.

## ✨ Features
- 🗺️ Live Crowd Heatmap — Prayagraj map with real-time density circles
- 📊 Live Crowd Density Dashboard — All zones with % capacity bars
- 📈 Hourly Crowd Prediction — ML-based forecast 6AM to 11PM
- 🧭 Route Optimization — 3 safe routes with crowd levels and timing
- ⚠️ Alert System — Critical, warning, and info alerts with ticker
- 🎛️ Simulation Engine — Normal/Snan/Festival/Emergency scenarios
- 📉 Analytics Dashboard — 7-day trend, peak forecast, key metrics
- 👮 Authority View — Police deployment, medical units, incident log
- 🤖 AI Chatbot — Real-time crowd guidance for pilgrims
- 📢 Broadcast System — Authority alert messaging

## 🛠️ Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- SVG-based interactive map
- AI-assisted development (Claude AI + Cursor)
- Deployed on Netlify
